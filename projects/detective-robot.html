Python Machine Learning Computer Vision

This project was for a team based ENPH project course that focuses on machine learning.

Learnings Summary

From this project, I learned how to train a TensorFlow neural network, how to use the OpenCV library, and generally gained better proficiency with Python.

Introduction

This project revolved around a simulated physics environment (gazebo) and developing a robot that was capable of navigating the environment by following road lines, and reading “clues” off of signs using a neural network. The two components of this project were the driving and the reading, of which I contributed to the reading.
